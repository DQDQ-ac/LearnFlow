# LearnFlow 视频学习助手

基于 Android + Spring Boot 的视频学习与任务管理系统

---

## 项目简介

LearnFlow 是一个用于视频学习管理的移动端应用，支持视频播放、学习记录、任务管理以及学习数据统计功能。

在日常使用视频进行学习的过程中，常见问题包括：

- 学习进度难以记录
- 重点内容无法重复强化学习
- 学习过程缺乏整理和回顾
- 无法直观了解学习情况

该项目的目标是将视频学习过程结构化，使学习行为可以被记录、追踪和回顾。

---

## 功能模块

### 视频学习

- 基于 ExoPlayer 实现视频播放功能  
- 支持播放进度记录与断点续播  
- 支持 A-B 区间循环播放，用于重点内容重复学习  
- 支持关键帧截图记录  

---

### 学习记录与笔记

- 支持学习笔记添加与查看  
- 自动记录学习时长与播放进度  
- 支持历史学习记录查询  

---

### 任务管理

- 支持学习任务的创建、编辑与删除  
- 任务可关联具体视频内容  
- 支持任务完成状态管理  

---

### 学习数据统计

- 学习热力图展示每日学习时长  
- 学习记录按时间维度统计  
- 支持学习情况回顾  

---

### 用户系统

- 用户注册与登录功能  
- 基于 Token 的身份认证  
- 用户数据与学习记录绑定  

---

## 核心流程

### 视频学习流程

1. 用户进入视频播放页面  
2. 使用 ExoPlayer 播放视频  
3. 系统记录播放进度与学习时长  
4. 支持 A-B 区间循环学习  
5. 学习记录同步至后端  

---

### 学习统计流程

1. 客户端上传学习记录  
2. 后端按日期进行统计  
3. 前端展示热力图与统计结果  

---

## 项目特点

- 前后端分离架构设计  
- Android 端基于 MVVM 架构  
- 使用 StateFlow 管理 UI 状态  
- RESTful API 规范设计  
- 学习行为数据结构化存储  
- 支持学习过程记录与回顾  

---

## 项目亮点

- 独立完成 Android 客户端 + 后端 + 数据库设计  
- 实现视频学习行为的完整闭环（学习 → 记录 → 统计 → 回顾）  
- 支持 A-B 区间循环播放，提升重点内容学习效率  
- 学习数据可视化（热力图展示）  
- 工程结构清晰，便于后续扩展
